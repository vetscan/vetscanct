- ServicesSearch (карточки с fallback, специальная карточка с градиентом)
- Prices
- OncologyHelp
- FAQ
- Team (карусель)
- Testimonials (карусель, круглые индикаторы)
- ContactForm
- Footer (градиент)
- AppointmentModal (Portal, центрирование)
- ThemeToggle (кастомный переключатель)

- Светлая тема по умолчанию
- Все CSS переменные с fallback значениями
- Адаптивный дизайн
- Анимации и переходы
- Моргающие точки статуса телефонов
