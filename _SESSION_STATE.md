## Что сделано в проекте:

### ✅ Основные компоненты
- Header (липкий, с двумя телефонами, кастомный дропдаун языка, мобильное меню)
- Navigation (с фоном, кнопка "Записаться на прием", адаптивная)
- Hero (бирюзовый градиент, легкая тень, адаптивный)
- ServicesSearch (карточки с fallback, специальная карточка с градиентом, адаптивная)
- Prices (адаптивные карточки, лапки справа)
- OncologyHelp (только текст, адаптивный)
- FAQ (адаптивные карточки вопросов)
- Testimonials (адаптивная карусель с выравниванием высоты)
- ContactForm (адаптивная форма)
- Footer (градиент, адаптивный, аккордеон услуг на мобильных)
- AppointmentModal (Portal, центрирование, адаптивный)
- ThemeToggle (кастомный переключатель)
- MapSection (адаптивная карта)
- PageShell (адаптивная обертка страниц)
- PricesServices (адаптивные карточки услуг)

### ✅ Фичи
- Светлая тема по умолчанию
- Все CSS переменные с fallback значениями
- Полная адаптация под все размеры экранов
- Анимации и переходы
- Моргающие точки статуса телефонов
- Водяные знаки на главной странице (адаптивные)
- ВСЕ размеры в px для стабильного отображения

## Технологии:
- Next.js 15 (App Router)
- JavaScript
- CSS Modules
- React Hooks
